DNS

Domain Name System (DNS) was created to be an open public communication protocol. Domain Name System (DNS) is an associated name such as www.robinhood.com with a corresponding address for the communication.

Device lookup for the domain name to get the associated IP address to connect the device to reach the destination of the internet.

DNS servers are connected in a distributed hierarchy by splitting the server such as .com, .net, .org, .gov, .edu.
